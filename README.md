# Bus Pass Registration System

A Spring Boot + Thymeleaf web application for managing bus pass registrations.
The system allows users to create, view, update, and delete bus pass records using
HTML, CSS, and JavaScript on the frontend, and Spring Boot with JDBC on the backend.

---

## Features

- Register a new bus pass
- View all bus passes
- View bus pass details
- Edit existing bus passes
- Delete bus passes
- Responsive UI using HTML and CSS
- Client-side enhancements using JavaScript
- Unit testing using JUnit 5 and Mockito

---
